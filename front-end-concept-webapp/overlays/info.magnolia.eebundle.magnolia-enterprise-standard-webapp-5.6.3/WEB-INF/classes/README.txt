Magnolia Enterprise Edition Standard pre-installed with several modules.

This package contains:

   Magnolia Core 5.6.2
   magnolia-empty-webapp 5.6.3
   magnolia-empty-webapp 5.6.3
   Magnolia Internationalization Framework 5.6.2
   Magnolia JAAS 5.6.2
   magnolia-license 1.6.1
   Magnolia Backup Module Project 2.2.1
   Magnolia Module Content Dependencies 1.8
   Google Sitemap Module 2.5
   magnolia-module-mail 5.5
   magnolia-module-rssaggregator 2.6
   magnolia-module-soft-locking 2.7
   Magnolia Rendering 5.6.2
   Magnolia Templating 5.6.2
   Magnolia Templating for JSP 5.6.2
